史景喆

Hi! 我是史景喆，此前是清华大学 交叉信息研究院（即姚班，由图灵奖得主姚期智院士创办）计算机科学与技术专业的本科生。

近期，我很荣幸有机会得到优秀教授的指导并与他们学习。目前，我正与普林斯顿的Mengdi Wang教授和Sanfeng Wu教授、斯坦福的James Zou教授，以及清华的Hang Zhao教授合作，研究方向包括LLM智能体、LLM推理、LLM4Science、Scaling Laws及其解释等。

我也很荣幸在参加物理竞赛时遇到了一群才华横溢的朋友，我们创办了CPHOS，为高中物理竞赛选手免费提供线上模拟竞赛平台。与在CPHOS遇到的一些优秀朋友，我们还开展了一些有趣的相关研究，包括CHOPS项目。

我很幸运能与我的高中和大学同学马钦伟合作。此前，我也很荣幸与Xiaolong Wang教授和Lei Li博士（目前在华盛顿大学担任博士后，我们在CPHOS相识）合作过。

我热爱学习物理并为物理竞赛做贡献。我在IPhO 2021中获得金牌（全球排名第10），并受邀担任IPhO 2022的线上阅卷员。

    Intrinsic Entropy of Context Length Scaling in LLMs
    Jingzhe Shi*, Qinwei Ma*, Hongyi Liu*, Hang Zhao^, Jeng-Neng Hwang, Lei Li^

    ICLR 2026
    Code / arXiv

  • 我们从内在空间（Intrinsic Space）的角度解释了上下文长度的缩放规律，通过自然语言和合成数据集的实验验证了理论假设和推导。
  • 我们发现从LLM中间状态测量的内在熵（Intrinsic Entropy）与下一个token预测损失呈线性关系，这可能是一个值得探索的有趣现象，并可能暗示了稀疏表示的成功。

    Scaling Law for Time Series Forecasting
    Jingzhe Shi*, Qinwei Ma*, Huan Ma, Lei Li

    NeurIPS 2024
    Code / arXiv / OpenReview

  • 我们提出了时间序列预测的Scaling Law理论框架，考虑了回看窗口以及数据集大小和模型大小。
  • 我们进行了实验来验证我们提出的理论和假设。
  • 我们的关键理论和实验发现是，最优回看窗口确实存在，并且随着数据集大小的增加而增加，这要求在提出新的时间序列预测模型时进行更公平的比较。

    		•
    CHOPS: CHat with custOmer Profile Systems for Customer Service with LLMs
    Jingzhe Shi, Jialuo Li, Qinwei Ma, Zaiwen Yang, Huan Ma, Lei Li

    COLM 2024
    Code / arXiv / OpenReview

  • 我们提出了CHOPS，一个LLM智能体，旨在通过结合小型和大型LLM来高效访问用户信息、与现有系统交互，并提供准确、安全的响应。使用我们在同一工作中提出的CPHOS数据集进行验证，CHOPS展示了增强或替代人工客户服务的潜力。

    		•
    Large Trajectory Models are Scalable Motion Predictors and Planners
    Qiao Sun, Shiduo Zhang, Danjiao Ma, Jingzhe Shi, Derun Li, Simian Luo, Yu Wang, Ningyi Xu, Guangzhi Cao, Hang Zhao

  • 我负责解码器部分。我利用DDPM在关键点空间中生成轨迹，以捕获未来轨迹的多模态分布。
